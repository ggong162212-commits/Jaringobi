# 자린고비 햄스터 🐹

목표 기간과 금액만 정하면 **하루 예산을 자동으로 나눠주고**, 매일 저녁 쓴 돈을 기록하면
아낀 만큼은 다음날로 적립되고 더 쓴 만큼은 다음날 예산에서 차감되는 귀여운 절약/예산 관리 모바일 웹앱입니다.

## 바로 실행하기

별도 빌드 과정이 없습니다. 정적 파일만 열면 됩니다.

```bash
# 1) 아무 정적 서버나 사용
python3 -m http.server 8000
# 또는
npx serve .

# 2) 브라우저에서 접속 (모바일 화면 권장)
open http://localhost:8000
```

`index.html` 파일을 더블클릭해 바로 열어도 동작합니다.

## 핵심 로직

- **하루 기본 예산** = `목표 금액 ÷ 목표 기간(일수)`
- **오늘 쓸 수 있는 금액** = `하루 기본 예산 + 지금까지 누적된 잔액(적립/초과 합산)`
- 매일 저녁 오늘 쓴 금액을 입력하면
  - `(오늘 쓸 수 있는 금액 − 실제 지출)` 이 **+** 면 → **적립**되어 다음날 예산에 더해짐
  - `(오늘 쓸 수 있는 금액 − 실제 지출)` 이 **−** 면 → **초과분**이 다음날 예산에서 차감됨

> 예) 30일 동안 300,000원 목표 → 하루 기본 예산 10,000원
> - 1일차: 10,000원 가능 → 8,000원 사용 → **+2,000원 적립**
> - 2일차: 10,000 + 2,000 = **12,000원** 가능 → 15,000원 사용 → **−3,000원 초과**
> - 3일차: 10,000 − 3,000 = **7,000원** 가능
>
> 누적 잔액이 매일 굴러가며 정확히 계산됩니다. (구현은 `app.js`의 `carryBefore` / `compute` 참고)

## 화면 구성

1. **설정** — 목표 기간·금액·시작일 입력 시 하루 예산을 즉시 미리보기
2. **홈** — 햄스터 마스코트, "오늘 쓸 수 있는 금액", 남은 기간·누적 적립·진행률
3. **지출 입력** — 오늘 쓴 금액 입력 → 적립/초과 결과를 햄스터 반응과 함께 표시
4. **기록/캘린더** — 날짜별 예산 대비 지출, 적립/초과 내역
5. **축하** — 목표 기간 종료 시 총 절약액 축하 화면 (컨페티)

## 기술

- 순수 HTML / CSS / Vanilla JS (프레임워크·빌드 없음, 가볍고 빠름)
- 로그인·서버 없음 — 모든 데이터는 브라우저 **localStorage** 에 저장 (새로고침해도 유지)
- 모바일 우선 세로 레이아웃, PC에서도 깨지지 않는 반응형
- 토스 스타일의 미니멀 UI + 따뜻한 크림/베이지 톤 + 포인트 컬러(주황)
- 햄스터 마스코트는 표정 3종(행복/평범/시무룩)의 인라인 SVG로 구현

### 마스코트 이미지 교체

마스코트는 `app.js`의 `hamsterSVG()` 함수에서 SVG로 그립니다.
실제 햄스터 사진(PNG)으로 바꾸려면 이미지를 프로젝트에 추가하고 해당 함수가
`<img>` 를 반환하도록 교체하면 됩니다.
